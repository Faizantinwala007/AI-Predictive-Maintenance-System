🤖 AI Predictive Maintenance System
📌 Overview

This project demonstrates an AI-based predictive maintenance system that analyzes machine conditions such as temperature and vibration to predict potential failures. The system helps industries reduce downtime, improve efficiency, and optimize maintenance scheduling.

🚀 Features
Machine Learning-based Failure Prediction
Data Analysis and Preprocessing
Real-time Prediction Capability
Accuracy Evaluation
Data Visualization (Graph Analysis)
